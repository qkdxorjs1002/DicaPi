### 하드웨어 (Hardware)
| Name         |  Description  |
| ------------ | :-----------: |
| RPi 3B+      |       -       |
| TFT-LCD      | RPi 3.5in(a)  |
| PiCamera     |    V1 5MP     |
| BMS & LiPo   |    5000mAh    |
| LED          |   Hi x 2ea    |
| PushButton   |       -       |
| PLA Filament | Made in Korea |
| Nut & Bolt   | 5mm x 4 pair  |
